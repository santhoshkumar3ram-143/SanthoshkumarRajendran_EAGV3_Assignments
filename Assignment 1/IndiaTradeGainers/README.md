# India Top Stocks Scanner - Chrome Extension 📈🇮🇳

A sleek, premium Chrome Extension designed for the Indian Stock Market. It instantly fetches and displays the Top 10 Gainers on the National Stock Exchange (NSE) in real-time, giving traders a quick market overview without needing to keep heavy charting software open.

## ✨ Features
- **Real-Time Data:** Pulls live data from TradingView's screener API.
- **Premium UI/UX:** Built with a beautiful dark mode and glassmorphism interface.
- **Micro-Animations:** Includes customized loading animations, hover states, and staggered list rendering.
- **No API Keys Required:** Uses Chrome's `host_permissions` manifest setting to bypass CORS restrictions efficiently.

## 🛠️ Installation (Developer Mode)

1. Download or clone this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right corner.
4. Click on **"Load unpacked"**.
5. Select the `IndiaTradeGainers` folder from the cloned repository.
6. The extension is installed! Pin it to your toolbar for quick access.

## 💻 Tech Stack
- Frontend: HTML5, CSS3 (Vanilla, custom design system)
- Logic: Vanilla JavaScript
- API: TradingView Scanner Endpoint
- Environment: Chrome Extension Manifest V3

## 📝 Usage Limitations
This extension relies on publicly available endpoints designed for the TradingView scanner. It is intended for educational purposes and personal trading workflows.
